关于这个协议

cuwcl4c:// 是一个用于方便盗链下载网络上的文件到指定文件而创建的协议。

该协议完全开放，任何人都可以提交改进方案。

cuwcl4c:// | 1 | http://example.com/dl.php?key=1234 | Photos.7z | http://dl.example.com/
协议头 分隔符 版本 完整地址 下载文件名 引用来源

其中，协议版本将内嵌至用户脚本及下载器。

下载器

目前官方只有使用bash脚本写的一个简单的下载器，并安装至/opt/cuwcl4c目录下。

依赖项分别为 bash、curl。其他的 Shell 可能能用，但是我没测试 :3

我的测试环境为 Linux Mint 17，Ubuntu 用户用起来应该没问题。

安装

在控制台执行下述脚本即可:

tmpDir=/tmp/cuw-setup
mkdir -p "$tmpDir" && cd "$_";
curl -O https://raw.githubusercontent.com/JixunMoe/cuwcl4c/master/downloader/installScript.sh \
-O https://raw.githubusercontent.com/JixunMoe/cuwcl4c/master/downloader/downloader.sh
chmod +x downloader.sh installScript.sh
sudo ./installScript.sh
cd ~/ && rm -r "$tmpDir"

安装过程中会询问您的 root 管理员密码 (改成 sudo 会好些?)
